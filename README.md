# Projeto de CRUD de usuários

## Tecnologias utilizadas

- Next.js
- TailwindCSS
- Shadcn/UI

## Dependências

- Java
- Node.js
- npm

## Instruções para execução

1. Clonar o repositório
2. Instalar as dependências
3. Executar o projeto

   ```bash
   git clone https://github.com/0thigs/at4.git
   cd at4
   npm install
   npm run dev
   ```
